--------------------------------tracker.py--------------------------------
* Για να θεωρείται οτι πούλησε θα πρέπει να υπάρχει η διεύθηνση και στα input και στα output
	Υπολογισμός πώλησης: Total_Spent = Total_Input(Address) - Total_Output(Address)

* Για να θεωρείται οτι αγόρασε θα πρέπει να υπάρχει η διεύθηνση μόνο στα output
	Υπολογισμός αγοράς: Total_Bought = Total_Output(Address)

* Σε περίπτωση που η συναλλαγή είναι μηδενική, τότε επιστρέφουμε None

--------------------------------scanner.py--------------------------------
* Συλλέγει τις τελευταίες συναλλαγές του CSV με όλα τα transactions

* Συγκρίνει τους τελευταίους (3 ή 4) αριθμούς των block του CSV αρχείου και αυτών που βρήκε το API (tracker.py)

* Εάν κάποιο block δεν υπάρχει στο CSV αλλά υπάρχει στο API τότε η συηνάρτηση update_csv προσθέτει την συναλλαγή στο αρχείο
